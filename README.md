# TaskFlow App 📱

Um aplicativo mobile completo de gerenciamento de tarefas desenvolvido em React Native com TypeScript. 
Este projeto implementa funcionalidades de ponta a ponta, incluindo navegação (Tabs e Stack), consumo de API, persistência local de dados, CRUD de tarefas, contextos globais e tipagem forte com suporte nativo a temas Dark e Light.

## 🚀 Funcionalidades
- **Autenticação Simples**: Acesso restrito com níveis de usuário (admin / comum).
- **CRUD Completo**: Crie, edite, exclua e visualize detalhes de cada tarefa.
- **Armazenamento Local**: Tudo é salvo utilizando AsyncStorage para você nunca perder suas anotações.
- **Integração com API**: Chamadas diárias a uma API de frases motivacionais (DummyJSON).
- **Temas**: Suporte a Modo Escuro (Dark Mode) e Claro (Light Mode) configuráveis no perfil.
- **Navegação Avançada**: O projeto usa React Navigation combinando navegação fluida em Abas (Bottom Tabs) e Pilhas (Stack).
- **Animações e Feedback**: Loadings detalhados, animações suaves e validação de informações.

## 🛠️ Tecnologias Utilizadas
- **React Native / Expo**
- **TypeScript**
- **React Navigation**
- **AsyncStorage**
- **Context API & Custom Hooks**

## ⚙️ Como executar o projeto
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor do Expo:
   ```bash
   npm start
   ```
3. Abra o app utilizando o Expo Go em seu dispositivo móvel ou através de um emulador Android/iOS.
